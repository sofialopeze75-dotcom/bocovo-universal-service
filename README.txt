▸ COMMENT MODIFIER
──────────────────
1. TEXTES : ouvrez le fichier .html concerné dans un éditeur.

2. COULEURS : ouvrez styles.css, modifiez les variables en haut :
     --navy        bleu marine (couleur principale)
     --gold        doré (couleur d'accent)
     --ivory       fond beige clair
     --cream       fond crème

3. LOGO / PHOTOS : remplacez les fichiers dans assets/
   en conservant les mêmes noms.

4. TÉLÉPHONE : recherchez "+229 01 61 31 50 72" dans tous les fichiers.

5. EMAIL : recherchez "contact@bocovo-universal-services.com".

6. TÉMOIGNAGES : modifiez les blocs <div class="testimonial"> dans index.html.

7. FAQ : modifiez les blocs <div class="faq-item"> dans index.html.

▸ FORMULAIRE DE CONTACT
───────────────────────
Le formulaire ouvre actuellement le client mail du visiteur (mailto:).
Pour un envoi automatique, intégrez l'un de ces services gratuits :
  • Formspree     → https://formspree.io
  • Web3Forms     → https://web3forms.com
  • EmailJS       → https://emailjs.com
  • Getform       → https://getform.io

Il suffit de remplacer l'attribut action="..." du <form>
ou de modifier la fonction dans script.js.

▸ PUBLICATION
─────────────
Uploadez TOUS les fichiers sur n'importe quel hébergeur :
  • Hébergeurs classiques : Hostinger, OVH, IONOS, o2switch
  • Hébergeurs gratuits   : Netlify, Vercel, GitHub Pages, Cloudflare Pages

▸ CARTE GOOGLE MAPS
───────────────────
La carte est dans contact.html. Pour changer la position,
modifiez les coordonnées dans l'URL de l'iframe :
src="https://www.google.com/maps?q=LATITUDE,LONGITUDE&z=14&output=embed"

▸ RÉSEAUX SOCIAUX
─────────────────
Les icônes Facebook et LinkedIn pointent vers "#" dans le footer.
Remplacez-les par vos vraies URL dans chaque fichier .html.
